Java

Java is a versatile programming language widely used in backend development.

It follows the Object-Oriented Programming (OOP) paradigm.

public class HelloWorld { }
  • Platform-independent
  • Used in Android development
Python

Python is known for its readability and simplicity.

It is widely used in data science and AI.

print("Hello, World!")
  • Interpreted language
  • Great for automation
JavaScript

JavaScript is essential for frontend and backend development.

It allows dynamic interactions on websites.

console.log("Hello, World!");
  • Runs in the browser
  • Node.js enables backend development
C++

C++ is a high-performance language used in game development.

It provides both low-level and high-level features.

#include <iostream>
  • Supports Object-Oriented Programming
  • Used in system programming
HTML & CSS

HTML structures web pages, while CSS styles them.

They form the foundation of frontend web development.

<h1>Hello World</h1>
  • HTML is for structure
  • CSS is for styling
